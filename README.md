# T2 NArFU Connect (прототип)

Данный проект является прототипом цифровой экосистемы для студентов и преподавателей САФУ, интегрированной в приложение оператора T2. Проект направлен на объединение разрозненных университетских сервисов в едином интерфейсе.

## Основные возможности:
- доступ к Sakai и расписанию: быстрый переход к учебным материалам и актуальному графику занятий;
- мгновенные уведомления: оперативное оповещение об изменениях в расписании (например, перенос аудитории);
- единый профиль: авторизация через студенческий аккаунт, связанный с тарифом T2;
- эксклюзивный тариф: безлимитный трафик на образовательные ресурсы университета.

## Начало работы:
- установка зависимостей:
  ```bash
  npm install
  ```
- запуск в режиме разработки:
  ```bash
  npm run dev
  ```
- сборка проекта:
  ```bash
  npm run build
  ```

## Структура проекта:
- `src/components/`: UI-компоненты приложения (дашборд, расписание, профиль);
- `src/data/`: моки данных для демонстрации функционала;
- `src/styles/`: глобальные стили и темы.
