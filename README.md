# T2 NArFU Connect

Цифровая экосистема для студентов и преподавателей САФУ, интегрированная в приложение оператора T2. Проект направлен на объединение разрозненных университетских сервисов в едином интерфейсе.

## Основные возможности:
- доступ к Sakai и расписанию: быстрый переход к учебным материалам и актуальному графику занятий;
- мгновенные уведомления: оперативное оповещение об изменениях в расписании (например, перенос аудитории);
- единый профиль: авторизация через студенческий аккаунт, связанный с тарифом T2;
- эксклюзивный тариф: безлимитный трафик на образовательные ресурсы университета.

## Технологический стек:
- React 19: современная библиотека для создания пользовательских интерфейсов;
- TypeScript: строгая типизация для обеспечения надежности кода;
- Vite: быстрый инструмент сборки проекта;
- Framer Motion: библиотека для реализации плавных анимаций и переходов;
- Lucide React: набор иконок для интерфейса.

## Начало работы:
- установка зависимостей:
  ```bash
  npm install
  ```
- запуск в режиме разработки:
  ```bash
  npm run dev
  ```
- сборка проекта:
  ```bash
  npm run build
  ```

## Структура проекта:
- `src/components/`: UI-компоненты приложения (дашборд, расписание, профиль);
- `src/data/`: моки данных для демонстрации функционала;
- `src/styles/`: глобальные стили и темы.
